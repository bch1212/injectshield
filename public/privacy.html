Privacy

We log scan events for usage metering and abuse detection. By default we store the first 500 characters of submitted text in a private audit log so you can review what was flagged.

No-logging mode

Pro-tier customers may enable no-logging mode on a per-key basis. When enabled, no input text is stored and no text is used for any training, evaluation, or model improvement.

What we collect

  • Email address (for key delivery and billing).
  • API key + tier metadata (in Cloudflare KV).
  • Per-request: timestamp, key, context, confidence, threat type, text length.
  • First 500 chars of input text (omitted entirely in no-logging mode).

What we do NOT do

  • Sell or share data with third parties.
  • Train models on customer text.
  • Store text or metadata for keys that have requested deletion (we delete within 7 days of request).
